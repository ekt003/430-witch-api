~ Project 1 API Documentation ★ ~

The site's purpose

My API lets you make, save, and export cute little 2D witch icons. It was refreshing to work on something a little more light hearted than my research studio project, which was the goal.

Part of the app handled by the API

The API handles all of the user information. This means that it stores user names and icons and then loads them back into the main canvas on request.

What Went Right

I like the art and I like the cutesy witch aesthetic I went with. I also actually made mockups for the site and stayed true to them, which rarely ever happens. I also wrestled with canvas which is something I hated in 330.

What Went Wrong

I hard core over committed time to my friends who were struggling this past week instead of working on this project. I have learned nothing and will do it again, but I figured that it would probably still be good to say.

What Would You Add

I would love to add stuff like colors and more customization in the future are well as a more robust information storage system with logins and the whatnot.

Above and Beyond

  • Added the ability to export images to PNGs
  • Created the ability for players to share their icons via links
